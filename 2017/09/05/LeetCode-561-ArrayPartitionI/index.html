「LeetCode」561 - Array Partition I

发表于 2017-09-05 | 分类于 leetcode | | 阅读次数
字数统计 467 | 阅读时长 2

题目描述

Given an array of 2n integers, your task is to group these integers into n pairs of integer, say (a1, b1), (a2, b2), …, (an, bn) which makes sum of min(ai, bi) for all i from 1 to n as large as possible.

Example

Note

  1. n is a positive integer, which is in the range of [1, 10000].

  2. All the integers in the array will be in the range of [-10000, 10000].

题目大意为给定一个长度为 2n 的整型数组, 需要我们将这个整形数组组成 n 对, (a1, b1), (a2, b2), …, (an, bn), 求将每一对数的最小值相加求和的最大整数。

n 为一个正整数, 范围在 1 到 10000 之间。数组中所有的元素范围在 -10000 到 10000 之间。

解题思路

我们先来看一下题目中给出的示例, 示例中的数组为 [1, 4, 3, 2], 那么就可以组成两对数。那么如何得到这两对数的最小值求和的最大结果呢?

我们可以先来暴力穷举一下, 注 min(1, 2) 和 min(2, 1) 视为同一结果, 因为对其求最小值结果相同。

最小值1 最小值2 结果
min(1, 2) = 1 min(3, 4) = 3 4
min(1, 3) = 1 min(2, 4) = 2 3
min(1, 4) = 1 min(2, 3) = 2 3

只要保证最小的两个数组成一对, 然后除去已组成队的数字继续寻找两个最小的元素组成一队, 如此循环直到全部组对完毕。

那么接下来我们的任务就很简单了:

  1. 对数组进行排序。

  2. 将索引能够被 2 整除的元素的值相加(因为排序后索引能够被 2 整除的数一定是一对数中较小的那个)。
